# Create React App + ChakraUI + React Redux + Typescript

Simple template CRA with ChakraUI + React Redux + Typescript

## How to start

```sh
> git clone https://github.com/toyamarodrigo/cra-chakra-redux-tsc.git

> cd cra-chakra-redux-tsc

> yarn

> yarn dev
```
